# Installing and Setting up Remote Lighting

Installation and setup includes several sets of tasks from physically installing the light bulbs into fixtures to programming network lighting groups.

Installation of your Remote Network Lighting kit includes installing the light bulbs into light fixtures, preparing the remote control, and programming lighting groups.

1. Install light bulbs.
2. Prepare remote control.
3. Program lighting groups.
